# Immune Cell Swarming Dynamics

## Overview
This repository contains code used to generate results for immune cell swarming dynamics in:
- Sterile conditions
- In presence of bacteria

Due to dependency on local MATLAB modules, only partial execution may be possible. However, plotting scripts and processed data are provided for reproducibility of figures.

---

## Contents

- MATLAB simulation files (main model; may require additional local dependencies)

- Python plotting scripts.

- ## Notes

- MATLAB simulation code depends on additional local files and may not run standalone.
- Provided data files are sufficient to reproduce all figures shown in the manuscript.
- Default datasets correspond directly to results used in the paper.

---

## Purpose

This repository is provided to ensure transparency and reproducibility of simulation results during peer review.
